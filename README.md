## Description
it is a React-based web application developed as part of a web development assignment.  
The project demonstrates core frontend concepts such as component-based architecture, state management using Context API, and dynamic UI updates. It provides a simple and interactive product listing interface with cart functionality and search support.

The application focuses on clean UI, modular code structure, and practical implementation of real-world e-commerce features.

---

## Tech Stack
- React (Vite)
- JavaScript (ES6+)
- React Router DOM (Client-side Routing)
- Context API (State Management)
- React Toastify (Notifications)
- HTML5 & CSS3

---

## Features
- Product listing with dynamic rendering
- Add to cart functionality
- Increase and decrease cart item quantity
- Remove items from cart
- Search products in real time
- Cart subtotal calculation
- Responsive and user-friendly UI
